# Tutorial sequence

## Location and purpose

This file applies to `docs/tutorial/`. This directory is organized as a progressive, numbered tutorial intended to introduce the package from first principles through advanced use.

## Contents at this level

- `001.intro.md`, `002.concept-of-it.md`, and `003.first-usage.md` — orientation and first-use material.
- `004.warm-up.md`, `005.some-examples.md`, and `006.more-advanced.md` — progressive practical material.
- `007.above-expectations.md`, `008.cheatsheet.md`, and `009.for-hackers.md` — advanced, quick-reference, and technically deep material.
- `AGENTS.md` — this contextual note.

## Current state and next reflections

The tutorial's nine-step path is a strong outline, but it should be checked against the eventual exported API and runnable examples. The most valuable next step is to make `003.first-usage.md` a complete verified example, then use it as the shared foundation for later pages and the cheat sheet.
