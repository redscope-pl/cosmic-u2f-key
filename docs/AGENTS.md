# Documentation hub

## Location and purpose

This file applies to `docs/`. This is the package documentation tree, combining an entry point, getting-started material, developer guidance, reference material, and a tutorial sequence.

## Contents at this level

- `AGENTS.md` — this contextual note.
- `README.md` and `GETTING_STARTED.md` — intended documentation entry points.
- `developer-guidance/` — material for contributors and maintainers.
- `reference/` — API or conceptual reference material.
- `tutorial/` — numbered learning content.
